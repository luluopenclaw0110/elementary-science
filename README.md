# 自然科學探索互動網頁

🌟 專為國小生設計的自然科學互動學習網站

## 五大分類

| 圖示 | 分類 | 內容 |
|-----|------|-----|
| 🌌 | 太空探索 | 太陽系、星座認知、月相變化 |
| 🦖 | 恐龍世界 | 恐龍種類、時間軸、滅絕之謎 |
| 🫀 | 人體奧秘 | 器官位置、消化旅程、循環系統 |
| 🌋 | 地球科學 | 天氣變化、火山構造、水循環 |
| 🌊 | 海洋世界 | 魚類與鯨豚、海底地形、珊瑚礁 |

## 功能特色

- ⭐ 每完成一個單元獲得星星
- 🏆 集滿一個分類的星星，解鎖成就徽章
- 🔇 安靜模式（靜音選項）
- 📱 響應式設計，支援手機和平板

## 技術架構

- 純靜態 HTML + CSS + JavaScript
- 無需後端伺服器
- 使用 GitHub Pages 部署

## 部署方式

網站已部署至 GitHub Pages：
https://luluopenclaw0110.github.io/elementary-science/

## 本地開發

```bash
# Clone repo
git clone https://github.com/luluopenclaw0110/elementary-science.git
cd elementary-science

# 使用任意靜態伺服器
python3 -m http.server 8000
# 或
npx serve .
```
